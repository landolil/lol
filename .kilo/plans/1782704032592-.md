## 4. FAA CFR PART 107 COMPLIANCE AND WAIVER ANALYSIS

### 4.1 Automatic Exemptions (Under 0.55 lbs / 250g)

| Rule | Requirement | ASB-UAS Status |
|---|---|---|
| §48 Registration | Required if ≥ 0.55 lbs | **EXEMPT** — under 250g (Category 1) |
| Remote ID | Required if registration needed | **EXEMPT** if under 249g on takeoff (no registration = no Remote ID required) |
| Category 1 Operations over people | No exposed rotating parts that lacerate | **Design target** — spiral body is enclosed; no exposed sharp blade edges |

### 4.2 Part 107 Rules That Apply (Research / Commercial / Educational)

Since this is for aeronautical research (non-recreational), **14 CFR Part 107** applies regardless of weight. A Remote Pilot Certificate (Part 107 exam) is needed. Under Part 107:

| Rule | What It Says | ASB-UAS Position |
|---|---|---|
| §107.51 — 100 mph limit | Max groundspeed 100 mph | Speed governor mandated; design target 45 mph normal, 100 mph governed maximum |
| §107.51 — 400 ft AGL limit | Max 400 ft above ground level | Cliff/updraft launches will exceed this → **WAIVER REQUIRED** from §107.51 |
| §107.31 — Visual Line of Sight | Must see it throughout flight | Swarm + wind drift → **WAIVER REQUIRED** from §107.31 |
| §107.35 — Single drone rule | 1 pilot, 1 drone at a time | 5-20 unit swarms → **WAIVER REQUIRED** from §107.35 |
| §107.29 — Daylight only | No night flight without anti-collision lights | Operate daytime only (sufficient for research) |
| §107.25 — No operation from moving vehicle | Must be stationary at launch | Hand launch from stationary position on cliff = **COMPLIANT** |

### 4.3 Waiver Strategy

Three waivers to apply for via FAA Aviation Safety Hub (previously FAADroneZone):

1. **§107.35 Waiver — Multiple sUAS Swarm Operations**
   - Request: Up to 20 units simultaneously under a single RPIC
   - Precedent: Multiple active waivers exist for 4-300 drones (confirmed from waiver database, 2026)
   - Key requirement: Each unit must have independent Flight Termination System
   - Key requirement: Aggregated C2 control system (ground station LoRa mesh)
   - Processing time: ~90 days typical

2. **§107.51 Waiver — Operations Above 400 ft AGL**
   - Justification: Aeronautical research in non-populated areas (ocean/canyon) per §89.120
   - Safety case: Remote locations, no overfly of people, NOTAM filed
   - Altitude ceiling requested: up to 1,200 ft AGL (or as approved)

3. **§107.31 Waiver — Beyond Visual Line of Sight (BVLOS)**
   - Justification: Wind drift in research scenario; LoRa GPS tracking at all times
   - Safety case: Over water/unpopulated areas, flight termination tether or parachute system

### 4.4 Speed Governor Requirement
- §107.51 prohibits operation above 100 mph groundspeed
- Device installs a **centrifugal speed governor** (passive mechanical) on spiral body rotation
- Electronic governor backup: flight controller monitors GPS groundspeed via telemetry and can trigger parachute/tether release if 95 mph exceeded

## 5. DEVICE CONCEPT: THE ARCHEMEDIAN SPIRALING BIRD UAS (ASB-1)

### 5.1 Design Philosophy

The ASB-1 is NOT a conventional multirotor drone. It is a **gyroscopically-stabilized autorotating spiral body** that:
- Looks and acts like a bird or seed pod in flight (maple seed / samara aerodynamics)
- Lifts by autorotation of its Archimedes spiral body in upward winds
- Self-steers into wind via gyroscopic precession (weathervane effect)
- Measures wind data passively (by reading its own rotation and drift)
- Transmits data via onboard LoRa radio back to ground station
- Returns via tether (primary keep-close system) or descends when released

### 5.2 Physical Specifications (Target)

| Parameter | Target Value | Rationale |
|---|---|---|
| Total mass | < 249g (0.549 lbs) | Below FAA 250g threshold; no registration/Remote ID required |
| Body diameter | 150mm max | Pocket-size for hand launch; fits in 6" tube launcher |
| Spiral length | 200mm | Provides enough blade area for autorotation at 5+ m/s wind |
| Number of spiral turns | 3 turns | Optimal per 2025 aerodynamic research (3-blade ASWT Cp 0.264) |
| Operating wind speed | 5–25 m/s (11-56 mph) | Cliffside/coastal launch environment |
| Normal operating altitude | 50–400 ft | Within standard Part 107 |
| Maximum altitude (waiver) | 1,200 ft AGL | Research waiver scenario |
| Governed max speed | 100 mph (governed) | §107.51 compliant with governor |
| Typical cruising speed | 20–45 mph | Wind-riding speed |
| Telemetry range | 3-8 km | LoRa 915 MHz air-to-air, 8 km air-to-ground |
| Sensor payload mass budget | 15g | Pressure probe + temp + humidity + IMU |
| Battery life | 90 min (passive autorotation), 30 min (motor-assist) | LiFePO₄ 2S 300mAh |

### 5.3 Propulsion Decision: Smart Rope vs. Fluid — Evaluation

The user asked to evaluate **Smart Muscle / Smart Rope actuation** vs. **internal fluid** for the spiral actuation.

**Decision Recommendation: Smart Rope / Smart Muscle wins for V1.0**

| Factor | Smart Rope (SMA/polymer) | Internal Fluid |
|---|---|---|
| Weight | ~4-6g for actuation | Reservoir = 20-40g (too heavy) |
| Complexity at Cypress University | Medium — soldering + braiding | High — hydraulic seals, pump, reservoir |
| Non-inductive compliance | Yes (dielectric braiding from CSMFAB0017) | Possible but fluid pump motor is inductive |
| Control | PWM servo driver or direct SMA current | Pump motor controller |
| Fabrication at university makerspace | 3D printed housing + SMA wire from hobby supplier | Requires CNC machining for seals |
| Reliability outdoors | High (no fluid leaks) | Lower (pressure leaks in cold/wind) |
| V1.0 recommendation | **USE THIS** | Defer to V2.0 |

**Smart Rope Implementation:**
- 4 SMA wires (nitinol, 0.1mm diameter, 45°C activation temp) control 4 spiral arm flex points
- Braided in UHMWPE dielectric braid per CSMFAB0017 (non-inductive)
- Controls pitch angle of spiral body during flight (passive + active modes)
- Total actuator mass: 4.2g

### 5.4 Keeping It Close — Tether Safety System

Problem: In strong cliff winds, the spiral will keep spiraling and ascending uncontrollably.

**Solution: Dual Containment System**

1. **Primary: Dielectric Tether**
   - 50m Dyneema (UHMWPE) braided tether, 1mm diameter
   - Mass: ~5g total (1mm × 50m = ~100mg/m × 50m)
   - Attaches to tail of device via swivel connector (allows free rotation)
   - Ground end: reel with electromagnetic brake (hand-controlled from ground box)
   - Tether adds effectively 0 drag to the spiral due to thin profile
   - Breakaway coupling at 150g tension (safety release if entangled)

2. **Secondary: Flight Termination System (FTS)**
   - Micro-parachute (4g, UHMWPE canopy, deploys at >400ft or via command)
   - LoRa command from ground station triggers parachute deployment
   - Meets FAA waiver requirement for independent FTS

3. **Tertiary: Speed Governor**
   - 3D-printed centrifugal flyweight governor on main spiral shaft
   - Physically increases pitch angle (braking effect) above 5000 RPM (~100 mph)
   - Passive mechanical — no electronics required, cannot fail electronically
